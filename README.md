# Python MCP Calculator

A simple calculator server built using the official Python SDK for the Model Context Protocol (MCP).

## Features

- Addition
- Subtraction
- Multiplication
- Division

## Requirements

- Python 3.10+
- MCP Python SDK

## Installation

```bash
pip install -r requirements.txt
```

## Run

```bash
python server.py
```

## Example Tools

- add
- subtract
- multiply
- divide
